Datasets

Here are handy links to all of the data sets used in the course that we can’t get through packages.

  • From The Portal Project Teaching Database
    • Portal_rodents_19772002_scinameUUIDs.csv [raw trapping data]
    • species.csv [species id metadata]
    • surveys.csv [trapping data by record id]
    • combined.csv [trapping data merged with record id metadata]
    • plots.csv [plot type metadata]
  • From Data Carpentry: Genomics Lesson
    • sequencing_results_metadata.txt [metadata from DNA sequencing]
  • From Data Carpentry: R for Data Analysis and Visualization of Ecological Data
    • survey_data_spreadsheet_messy.xls [example of untidy data]
  • From Software Carpentry: R for Reproducible Scientific Analysis
    • continents.RDA [gapminder continent-level data]
    • wide_eg.csv [blood albumin levels]
    • stocks.tsv [restaurant stock data]
    • gapminder.csv [country characteristics by year]
  • From ANGUS: Analyzing High Throughput Sequencing Data
    • rnaseq.zip [zipped folder of RNA sequence data]
  • From Ryan Peek’s Data wRangling for Ecologists course
    • 2001_mauna_loa_met_data.zip [meteorological station data from Mauna Loa]


Course content

For this course, we embrace the spirit of collaborative lesson development. We leverage the community’s effort to develop effective approaches to teach this material using modern educational psychology and instructional design, and draw lessons from the following open sources:

  • Software Carpentry: R for Reproducible Scientific Analysis
  • Data Carpentry: R for Data Analysis and Visualization of Ecological Data
  • Data Carpentry: Data Organization in Spreadsheets
  • Data Carpentry: Genomics Lesson
  • ANGUS: Analyzing High Throughput Sequencing Data; Exploratory RNAseq data analysis using RMarkdown materials
  • Ethan White and Zack Brym’s Data Carpentry for Biologists course
  • Ryan Peek’s Data wRangling for Ecologists course
  • Jenny Bryan’s Stat545 course
  • Chester Ismay and Albert Y. Kim’s Modern DiveR book
  • Michael Koontz’s Introduction to R course
