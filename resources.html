Course content

For this course, we embrace the spirit of collaborative lesson development. We leverage the community’s effort to develop effective approaches to teach this material using modern educational psychology and instructional design, and draw lessons from the following open sources:

  • Software Carpentry: R for Reproducible Scientific Analysis
  • Data Carpentry: R for Data Analysis and Visualization of Ecological Data
  • Data Carpentry: Genomics Lesson
  • Ethan White and Zack Brym’s Data Carpentry for Biologists course
  • Ryan Peek’s Data wRangling for Ecologists course
  • Jenny Bryan’s Stat545 course
  • Chester Ismay and Albert Y. Kim’s Modern DiveR book
  • Michael Koontz’s Introduction to R course
